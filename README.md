# Next-Documentation

This is my overengineered documentation repository based on:

* NextJS 14
* TurboRepo
* Markdoc
* tailwind w/ shadcn ui
* Orama Search

## Requirements

* Node 20
* pnpm 8+

## Installation

```
git clone git@github.com:noxify/next-documentation.git
cd next-documentation
pnpm i
```
